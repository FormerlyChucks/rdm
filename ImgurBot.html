ImgurBot

About

This bot reposts things from Reddit to Imgur. Those silly Redditors always talk about how Instagram steals their submissions, but they never mention that Imgurians do the same. I guess it shows that we suck all around, huh?

Setup

You'll need a Linux distro. Any distro that supports Python, Nano, Pip3 and Git will work. It might work on Windows/Mac, but I haven't tested that out.
Download the bot:
git clone https://github.com/Mr-Steal-Your-Script/ImgurBot
Change into its directory: 
cd ImgurBot
Install the needed packages: 
pip3 install -r requirements.txt

Getting API Keys

Create an Imgur app
- Go to /oauth2/addclient
- Fill in the form and get your Imgur client ID.
 Create a Reddit app
- Go to /prefs/apps
- Select "are you a developer? create an app..."
- Name it whatever you want to, but select "script"
- Fill in the rest of the form

Edit the configuration file:

nano config.py
- Edit "c_id" with the "personal use script"
- Edit "c_s" with the secret
- Edit "u_a" with the bot's description
- Edit "un" with the bot's username
- Edit "pw" with your bot's password
- Edit "i_id" with your Imgur client id
- For the list of subs, add some subs that you wish to repost from
Save the file
(ctrl x and Y)

Run the bot:

python3 main.py
