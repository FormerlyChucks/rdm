Python Forum


A Forum, made with Python
The goal is that you should have your own textboard/forum by the end of this guide. You should have Git, Pip and Python installed.

Features

- Dark Mode
- Board Creation
- Thread and Comment Creation
- No Tracking
- Easy to Setup

Setup/Run


This should be easy to set up.

Simply clone the repo:
git clone https://github.com/titletrack/Forum

Change into the directory:
cd Forum

Install requirements:
pip3 install -r requirements.txt

Actually run it:
python3 app.py
Your terminal should give you something like this: 

    INFO:     Uvicorn running on http://0.0.0.0:8000 (Press CTRL+C to quit)

Customization

In your browser, navigate to the http site.
You should have a white screen that says "Be the first to create a board?". Tap the "New Board" button to create your first board!. "Slug" is the board's slug, "Title" is the board's title. You will also have to create a [f]irst post. To make a board called "awesome" that links from /a/, your slug would be "a" and your title would be "awesome". After creating your board, go back to the main page. To edit the "[this is a footer]" message, open /templates/footer/html and edit the message. To edit the "[name]", you will need to adjust line 7 in /templates/base.html and line 22 in /templates/common.html. To edit "[logo]", do the same for line 3 in /templates/header.html.
That should be all! If you have any problems, create an issue in the GitHub repository.
