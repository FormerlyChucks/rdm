KarmaBot

About

This bot reposts things from the past, in hopes of gaining more karma.
By peeking at the source code, you can tell how it works:
- Go to /r/random
- Grab random post from /top
- If the submission is hosted on i.redd.it or i.imgur.com:
- Upload to imgur via url
- For the description, leave a short description about the bot
- Submit to subreddit with link to repost

Setup

Original Version

You'll need a Linux distro. Any distro that supports Python, Nano, Pip3 and Git will work.
Download the bot and change into its directory: 
git clone https://github.com/Mr-Steal-Your-Script/KarmaBot && cd KarmaBot/old
Install the needed packages: 
pip3 install -r requirements.txt
Create a reddit app
- Go to /prefs/apps
- Select "are you a developer? create an app..."
- Name it whatever you want to
- Select "script"
- For the description, leave a short description about the bot
- Leave "about url" blank
- For the "redirect uri", put a link to your favorite porno
Create an imgur app
- Go to /oauth2/addclient
- Name it whatever
- For the "Authorization callback URL:", put a link to your favorite porno
- Fill in your email address
- Leave a short description about the bot
- Prove that you aren't a robot
Edit the configuration file: 
nano config.py
- Edit "C_ID" with the "personal use script"
- Edit "C_S" with the secret
- Edit "U_A" with the bot's description
- Edit "UN" with the bot's username
- Edit "PW" with your bot's password
- Edit "I_ID" with your imgur client id
Save the file
(ctrl x and Y)
Run the bot: 
python3 main.py
